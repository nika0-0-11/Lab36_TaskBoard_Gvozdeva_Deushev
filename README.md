## Описание проекта

Веб-приложение для управления задачами. Оно позволяет пользователю создавать, просматривать, редактировать, отмечать выполненные и удалять задачи.

## Как запустить проект 

1. Скопировать проект и открыть его в **Visual Studio Code**:

    - Выполнить следующий код в **Git Bash**:

        ```bash
        git clone https://github.com/nika0-0-11/Lab36_TaskBoard_Gvozdeva_Deushev
        
        cd Lab36_TaskBoard_Gvozdeva_Deushev

        code .
        ```

2. Чтобы запустить бэкенд:

    - Выполнить следующий код в **Git Bash**:

        ```bash
        ### перейти в папку бэкенда
        cd TaskBoardApi
        
        ### Сборка
        dotnet build

        ### Запуск
        dotnet run
        ```

3. Чтобы запустить фронтенд:

    - Открыть `index.html` используя **Live Server**

## Главные выводы

1. **Модульность** - вынос `fetch` в `api.js` отделяет логику сети от интерфейса, код становится чище. 
2. **Асинхронность** - `async/await` не блокирует интерфейс при запросах к БД или серверу.
3. **Автомиграции** - `db.Database.Migrate()` сам создает/обновляет БД при запуске, не нужно ручных команд.
4. **Локальное состояние** - хранение `allTasks` в браузере дает мгновенную фильтрацию без лишних запросов 
5. **Безопасность** - `escapeHtml()` защищает от XSS-атак, заменяя спецсимволы в пользовательском вводе.
